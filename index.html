- About Me -

I'm an Electronic Engineer, specialized in Power Electronics.

Master degree at the University of Padova and currently employed as R&D Hardware engineer at Recom Power. Interested in Power Electronics and in developing firmware applications.

Power Electronics: DC motor electric drive

Design of a board to implement a current control in a low power DC motor. The control has been performed using a STM32 connected by means of external pins. The measure of the current, instead, has been performed using another external board ACS712. The controller used is a digital PID controller implemented with an algorithm in the micro controller. The circuit is capable to drive the motor in two different directions, thanks to the presence of a bidirectional H-bridge.

Power Electronics: CF-DAB prototype

Design and implementation of a Current Fed Dual Active Bridge for a 10kW battery charger application. Design of the magnetics and test of the overall topology. The firmware has been realized using TI microcontroller TMS320F28379D.

Firmware developement: ESP32_2432S028 Display

ESP32 programmed to display data fetched from a web site. The web site is used to read data associated to a solar power plant.
The library is capable to entablish a connection with the server, to login and to fetch data which are then displayed in the screen. For more informations, please visit README.md
